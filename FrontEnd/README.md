# Getting Started
  
To install npm modules
  
```
  npm install
```
  
To install ng2 charts
  
```
  npm install ng2-charts
  
  npm install ng2-charts@2.2.3 --save
``` 
  
To start the project
  
```
  ng serve
```
  
To access project homepage
  
```
  localhost:4200
```
